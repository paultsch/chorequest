Rue — Mascot Direction Review

4 distinct character directions for Rue, the Pyrch owl AI. Each explores a different personality axis. Review at the top of each card, then see how Rue looks in real UI contexts before deciding.

Brand brief: warm, wise, family-friendly Palette anchor: amber eyes #F59E0B Must work at 16px → 512px
OPTION A

Friendly Rue

Round. Warm. Immediately loveable.

The Duolingo play. Large round head, massive amber eyes, soft ear tufts, gentle color palette. Works for every age — kids adore it, parents trust it. This is the safe choice that could also become iconic with consistency.

OPTION B

Wise Rue

Knowing. Trustworthy. Quietly confident.

The brow ridges do all the work — they turn a cute owl into a wise one. This Rue feels like the advisor you actually want running your household. Parents see 'intelligence.' The slightly hooked beak and wing detail signal 'I know what I'm doing.'

OPTION C

Icon Rue

Bold. Graphic. Unmistakable at 16px.

Forget feathers — this is pure geometry. Navy circle, cream face, two giant amber eyes. It's the owl as a logo mark. Works on app store icons, favicons, embossed on merchandise, printed at 1cm. The boldness makes it memorable. This is the Apple SF Symbol approach: clarity over decoration.

RECOMMENDED
OPTION D

Warm Rue

Illustrated. Expressive. Full of heart.

The most character. Cheek blush. Feather texture. Implied smile. This Rue has a soul. It's the version that belongs in a children's book and also on a parent's phone. The purple/violet accent separates it from standard 'blue tech AI' and leans into something warmer and more distinctive. This is the north star for Pyrch as a family brand.

How to decide

  • •  If Rue is primarily a parent tool → Option B (Wise) or Option C (Icon)
  • •  If Rue needs to work for kids AND parents → Option A (Friendly) or Option D (Warm)
  • •  If app-store icon quality matters most → Option C (Icon) at all sizes
  • •  If you want a full character system later → Option D (Warm) is the only one built for that
  • •  Marketing pick → Option D. The cheek blush is the detail people share on social.
